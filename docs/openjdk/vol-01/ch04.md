# 第4章：init.cpp 全局初始化

> 本章讲解 `Threads::create_vm` 阶段 6 的核心——`init_globals()`（`src/hotspot/share/runtime/init.cpp`）。30 个子函数依次初始化 JMX、字节码、代码缓存、CPU 特性、Java 堆、Metaspace、解释器、模板表、SystemDictionary、编译器等核心子系统，把 JVM 从"空壳"变成"能跑字节码的运行时"。
>
> `init_globals()` 与 `vm_init_globals()`（已在 [3.5](#/openjdk/vol-01/ch03/05-os-init2)、[3.6](#/openjdk/vol-01/ch03/06-main-thread-create) 讲完）是两轮初始化：前者由 VM 线程执行 7 项基础设施，后者由 Java 线程执行 30 项业务系统。
