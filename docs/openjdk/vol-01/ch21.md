# 第21章：类型系统的创世纪——TypeArrayKlass 与核心类偏移

> 本章讲解 universe2_init() + javaClasses_init()：Universe::genesis() 创建 8 种 TypeArrayKlass（boolean/char/float/double/byte/short/int/long）并加载 vmSymbols；javaClasses_init() 计算 29 个核心类（java.lang.Object/Class/String/Thread 等）的实例字段偏移量——这些偏移是 JVM 后续访问 Java 对象字段的唯一途径。
