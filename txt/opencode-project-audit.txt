一、项目概述
────────────────────────────────────────────────────────────────────────────
基于 docsify 的个人技术博客，托管于 GitHub Pages，专注于 Java 并发编程。
使用纯静态 HTML + docsify 动态渲染，无构建工具（无 package.json/npm）。
核心逻辑集中在单一 JS 文件和单一 CSS 文件中。

三、关键架构特征
────────────────────────────────────────────────────────────────────────────
1. 双入口设计：根 index.html (含书签+分析) 与 docs/index.html (含 Hero+文楷字体)
2. 所有 JS 功能集中在单个插件函数中 (common.js:184-1218)
3. 所有 CSS 集中在单文件中 (common.css)，页面特有样式通过内联 <style> 覆盖
4. 自定义全文搜索（非 docsify-search 插件）
5. _sidebar.md 仅含首页链接，文章链接在 README.md 内容中硬编码
6. 无构建工具，所有 CDN 依赖直接在 HTML 中引用
